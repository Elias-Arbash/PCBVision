# Contributing Guide

Thank you for improving this project! To get started:

1. Fork the repository and create a feature branch.
2. Run the test suite with `python -m pytest`.
3. Follow the commit guidelines: use conventional commit messages where possible.
4. Include documentation updates alongside code changes.

## Development Environment

- Python 3.10+
- Install dependencies via `pip install -e .[dev]`
- Format code with `black` and lint with `ruff` (configured via pre-commit).

Please include tests when fixing bugs or adding features. For significant proposals open an issue first.
